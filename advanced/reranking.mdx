---
title: "Reranking"
icon: "arrow-up-narrow-wide"
description: "A smarter second pass that reorders results by true relevance."
---

Vector search is fast but rough. It casts a wide net and ranks only by embedding similarity. A reranker re-reads the top candidates against the query and reorders them by true relevance.

<Note>Use it when a wrong answer is expensive and precision matters more than speed. Skip it when latency is critical or results are already good.</Note>

## How to use it

Retrieve wide, rerank down:

<CodeGroup>
```python SDK
result = rag.retrieve(
    "How do I authenticate?",
    top_k=20,
    use_rerank=True,
    rerank_top_k=5,
)
```

```bash CLI
ragrails retrieve "How do I authenticate?" \
  --vector-db qdrant --collection docs --url http://localhost:6333 \
  --provider voyage --model voyage-3 \
  --top-k 20 --rerank --rerank-top-k 5
```

```bash REST API
curl -X POST http://127.0.0.1:8000/v1/retrieve \
  -H "Content-Type: application/json" \
  -d '{"query": "How do I authenticate?", "provider": "voyage", "model": "voyage-3", "vector_db": "qdrant", "collection": "docs", "url": "http://localhost:6333", "top_k": 20, "use_rerank": true, "rerank_top_k": 5}'
```
</CodeGroup>

<Tip>The reranker needs candidates to choose from, so set `top_k` higher than `rerank_top_k`.</Tip>

## Choosing a reranker

| Provider | Cost | Best for |
| --- | --- | --- |
| `voyage` | API call per query | Semantic relevance; catches matches with no shared keywords |
| `bm25` | Free, local, no network | Exact-term matching; offline or cost-sensitive setups |

<CodeGroup>
```python SDK
result = rag.retrieve("...", use_rerank=True, reranker=rag.reranker(provider="bm25"))
```

```bash CLI
ragrails retrieve "..." \
  --vector-db qdrant --collection docs --url http://localhost:6333 \
  --rerank --reranker bm25
```

```bash REST API
curl -X POST http://127.0.0.1:8000/v1/retrieve \
  -H "Content-Type: application/json" \
  -d '{"query": "...", "vector_db": "qdrant", "collection": "docs", "url": "http://localhost:6333", "use_rerank": true, "reranker": "bm25"}'
```
</CodeGroup>
