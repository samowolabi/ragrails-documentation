The basic pipeline works without any of this. Reach for an advanced feature when you hit the matching problem.

| Problem | Feature |
| --- | --- |
| Top results are off-target | [Reranking](/advanced/reranking) |
| Chat follow-ups return nothing | [Query rewriting](/advanced/query-rewriting) |
| Long chats slow down, drift, or cost too much | [Chat tuning](/advanced/chat-tuning) |
| The bot answers even when the index has nothing | [Chat tuning → answer quality](/advanced/chat-tuning#answer-quality) |
| Large crawls or API pulls partly fail | [Resilient ingestion](/advanced/resilient-ingestion) |
| You want a provider Ragrails doesn't ship | [Custom providers](/advanced/custom-providers) |
| The chatbot needs to act, not just answer | [Agentic tools](/advanced/agentic-tools) |
| Token spend is too high | [Cost optimization](/advanced/cost-optimization) |
