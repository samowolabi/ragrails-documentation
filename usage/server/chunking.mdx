## POST /v1/chunk

<CodeGroup>
```json Request
{
  "markdown": [
    {"id": "doc-1", "text": "# Guide\n\nContent.", "source": "files/guide.pdf", "metadata": {}}
  ],
  "chunk_size": 2000,
  "chunk_overlap": 200,
  "min_chunk_length": 100
}
```

```json Response
{
  "inputs": 1,
  "chunks": 4,
  "items": [...],
  "failed": 0,
  "errors": []
}
```
</CodeGroup>

`markdown` accepts a string, list of strings, or list of document dicts with a `text` field.
