description: "Store, edit, and delete chunks over HTTP."
---

## POST /v1/store

<CodeGroup>
```json Request
{
  "embedded_chunks": [...],
  "vector_db": "qdrant",
  "collection": "docs",
  "url": "http://localhost:6333",
  "batch_size": 64
}
```

```json Response
{
  "inputs": 10,
  "stored": 10,
  "failed": 0,
  "provider": "qdrant",
  "collection": "docs",
  "errors": []
}
```
</CodeGroup>

---

## POST /v1/edit

```json
{
  "chunks": [
    {"id": "chunk-id", "text": "Updated text.", "source": "files/guide.pdf", "metadata": {}}
  ],
  "provider": "voyage",
  "model": "voyage-3",
  "vector_db": "qdrant",
  "collection": "docs",
  "url": "http://localhost:6333"
}
```

---

{
