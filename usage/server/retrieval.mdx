## POST /v1/retrieve

```json
{
  "query": "How do I authenticate?",
  "provider": "voyage",
  "model": "voyage-3",
  "vector_db": "qdrant",
  "collection": "docs",
  "url": "http://localhost:6333",
  "top_k": 10
}
```

With reranking:

<CodeGroup>
```json Request
{
  "query": "How do I authenticate?",
  "provider": "voyage",
  "model": "voyage-3",
  "vector_db": "qdrant",
  "collection": "docs",
  "url": "http://localhost:6333",
  "top_k": 20,
  "use_rerank": true,
  "reranker_provider": "voyage",
  "reranker_model": "rerank-2-lite",
  "rerank_top_k": 5
}
```

```json Response
{
  "query": "How do I authenticate?",
  "search_query": "How do I authenticate?",
  "retrieved": 5,
  "items": [
    {
      "id": "chunk-id",
      "score": 0.87,
      "text": "...",
      "metadata": {"title": "Auth Guide"},
      "rerank_score": null
    }
  ],
  "failed": 0,
  "errors": []
}
```
</CodeGroup>
