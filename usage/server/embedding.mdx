## POST /v1/embed

<CodeGroup>
```json Request
{
  "chunks": [
    {"id": "chunk-1", "text": "Embedding converts chunks into vectors.", "source": "playground://embedding", "metadata": {}}
  ],
  "provider": "voyage",
  "model": "voyage-3",
  "input_type": "document",
  "batch_size": 64
}
```

```json Response
{
  "inputs": 1,
  "embedded": 1,
  "items": [
    {"id": "chunk-1", "text": "...", "embedding": [...], "source": "...", "metadata": {}}
  ],
  "failed": 0,
  "errors": []
}
```
</CodeGroup>
