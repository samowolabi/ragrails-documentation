This page is kept for existing links. The REST documentation now uses [REST Extraction](/usage/server/extraction) for `POST /v1/ingest/url`, `POST /v1/ingest/docs`, document uploads, and `POST /v1/ingest/api`.

Batch API ingestion with `apis` is SDK-only for now. The REST schema currently requires `url`, so `/v1/ingest/api` examples use one endpoint per request.

Continue with [REST Extraction](/usage/server/extraction).
