---
title: "SDK Embedding"
description: "Generate vectors for chunk files and write them to a vector store."
---

Embedding reads chunk JSON files and uses an embedding provider before writing vectors to a vector database.

```bash
pip install "ragrails[voyage,qdrant]"
```

```python
from ragrails import RagRails

result = RagRails().embed(
    input_dir="files/output/chunks",
    vector_db="qdrant",
    collection="rag_chunks",
    url="http://localhost:6333",
    files=None,
    batch_size=64,
    embedder="voyage",
    model="voyage-3",
)
```

Use `files` to embed only selected chunk files:

```python
result = RagRails().embed(
    input_dir="files/output/chunks",
    files=["guide.json"],
    vector_db="qdrant",
    collection="rag_chunks",
)
```

| Parameter | Purpose |
| --- | --- |
| `input_dir` | Directory containing chunk JSON files. |
| `vector_db` | `qdrant`, `pinecone`, or `weaviate`. |
| `collection` | Collection, index, or class name. |
| `url` | Optional vector database URL. |
| `files` | Optional list of chunk files to embed. |
| `batch_size` | Chunks per provider/storage batch. |
| `embedder` | Embedding provider. |
| `model` | Embedding model name. |
