Embedding converts chunks into vectors. With an embedding provider configured on `RagRails`, `embed()` works without any extra setup.

```bash
pip install "ragrails[voyage]"
```

```python
from ragrails import RagRails

rag = RagRails(embedding={"provider": "voyage", "model": "voyage-3"})

result = rag.embed(chunks=chunk_result.items)
```

<CodeGroup>
```python Request
result = rag.embed(chunks=chunk_result.items, batch_size=64)
```

```python Response
result.embedded  # 64
result.items[0]  # {
                 #   "id": "guide-0",
                 #   "text": "Ragrails builds modular RAG workflows...",
                 #   "embedding": [0.0132, -0.0719, 0.0241],  # 1024 floats
                 #   "metadata": {"title": "Guide", "chunk_index": 0},
                 # }
```
</CodeGroup>

## Override the embedder per call

`embed()` creates the embedder from the constructor `embedding` config. Pass an explicit `embedder` to override:

```python
result = rag.embed(
    chunks=chunk_result.items,
    embedder=rag.embedder(provider="voyage", model="voyage-3-large"),
)
```

## `embedder()` parameters

| Parameter | Default | Description |
| --- | --- | --- |
| `provider` | constructor default, else `"voyage"` | Embedding provider |
| `model` | constructor default, else `"voyage-3"` | Model name |
| `input_type` | `"document"` | `"document"` for indexing, `"query"` for retrieval |
| `options` | `None` | Provider-specific options dict |

## `embed()` parameters

| Parameter | Default | Description |
| --- | --- | --- |
| `chunks` | required | List of chunk dicts; each needs `id` and `text` |
| `embedder` | inherits constructor embedding | Embedder object; auto-created when omitted |
| `input_type` | `"document"` | Used when `embed` creates the embedder |
| `batch_size` | `64` | Chunks per embedding API request |

## Supported providers

| Provider | Extra | Models |
| --- | --- | --- |
| Voyage | `ragrails[voyage]` | `voyage-3` (default), `voyage-3-lite`, `voyage-3-large` |

<Note>Bring your own provider with `register_embedder()`. See [extending providers](/reference/sdk#extending-providers).</Note>
