---
title: "SDK Retrieval"
description: "Search stored chunks from Python."
---

Retrieval embeds the query, searches the vector database, and can optionally rerank the candidate chunks.

```python
from ragrails import RagRails

result = RagRails().retrieve(
    query="How do I configure the API client?",
    vector_db="qdrant",
    collection="rag_chunks",
    top_k=10,
    embedder="voyage",
    model="voyage-3",
    rerank=True,
    reranker="voyage",
    reranker_model="rerank-2-lite",
    rerank_top_k=5,
)
```

| Parameter | Purpose |
| --- | --- |
| `query` | User question or search text. |
| `vector_db` | Vector database provider. |
| `collection` | Collection, index, or class name. |
| `url` | Optional vector database URL. |
| `top_k` | Number of vector candidates. |
| `rerank` | Whether to rerank candidates. |
| `rerank_top_k` | Number of reranked results to return. |
