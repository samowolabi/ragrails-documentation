---
title: "CLI Overview"
description: "Run Ragrails stages from the terminal."
---

The CLI is best for terminal workflows, local automation, and CI jobs.

```bash
pip install ragrails
ragrails --help
```

| Stage | Commands | Details |
| --- | --- | --- |
| Ingestion | `setup-url`, `scrape`, `parse`, `fetch` | [Ingestion](/usage/cli/ingestion) |
| Chunking | `chunk`, `chunk-file` | [Chunking](/usage/cli/chunking) |
| Embedding | `embed` | [Embedding](/usage/cli/embedding) |
| Storing | `store` | [Storing](/usage/cli/storing) |
| Retrieval | `retrieve` | [Retrieval](/usage/cli/retrieval) |
