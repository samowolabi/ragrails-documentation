---
title: "CLI Ingestion"
description: "Ingest URLs, documents, and APIs from the terminal."
---

## URL setup and scraping

```bash
pip install "ragrails[url]"
ragrails setup-url --browser chromium
ragrails scrape https://example.com --mode full --output-dir files/output/web_crawled
```

## Documents

```bash
ragrails parse --folder files/input --output-dir files/output/docs
```

Parse specific files:

```bash
ragrails parse --files guide.pdf --files pricing.csv --input-dir files/input --output-dir files/output/docs
```

## REST APIs

```bash
ragrails fetch https://api.example.com/v1/products \
  --title Products \
  --header "Authorization:Bearer TOKEN" \
  --param limit:100 \
  --output-dir files/output/api
```

| Command | Purpose |
| --- | --- |
| `ragrails setup-url` | Install browser support for URL ingestion. |
| `ragrails scrape` | Scrape one or more URLs into markdown. |
| `ragrails parse` | Parse local documents into markdown. |
| `ragrails fetch` | Fetch an API response into markdown. |
