---
title: "MCP Ingestion"
description: "Use MCP tools to ingest URLs, documents, and APIs."
---

MCP ingestion tools create markdown files for later chunking.

| Tool | Purpose |
| --- | --- |
| `ragrails_setup_url` | Install browser support for URL ingestion. |
| `ragrails_scrape_url` | Scrape one or more URLs into markdown. |
| `ragrails_parse_documents` | Parse local documents into markdown. |
| `ragrails_fetch_api` | Fetch a REST API response into markdown. |

Example agent request:

```text
Use ragrails_scrape_url to crawl https://example.com in full mode and write markdown to files/output/web_crawled.
```

Use local paths that the MCP server process can access.
