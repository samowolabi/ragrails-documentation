---
title: "MCP Retrieval"
description: "Use MCP tools to retrieve stored chunks."
---

`ragrails_retrieve` searches a configured vector database and can rerank results.

Example agent request:

```text
Use ragrails_retrieve for the query "How do I configure Ragrails?" against qdrant collection rag_chunks with top_k 10. Rerank with voyage and return 5 results.
```

The tool returns ranked chunks and metadata that the AI client can use as context.
