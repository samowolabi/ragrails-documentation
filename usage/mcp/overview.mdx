---
title: "MCP Overview"
description: "Use Ragrails as local MCP tools for AI clients and agents."
---

The MCP server is best for AI clients and agents that support Model Context Protocol. Ragrails v1 documents local stdio MCP only. Hosted or HTTP MCP support is future work.

```bash
ragrails-mcp
```

| Stage | Tools | Details |
| --- | --- | --- |
| Setup | client configuration | [Setup](/usage/mcp/setup) |
| Ingestion | `ragrails_setup_url`, `ragrails_scrape_url`, `ragrails_parse_documents`, `ragrails_fetch_api` | [Ingestion](/usage/mcp/ingestion) |
| Chunking | `ragrails_chunk_directory`, `ragrails_chunk_file` | [Chunking](/usage/mcp/chunking) |
| Embedding | `ragrails_embed_chunks` | [Embedding](/usage/mcp/embedding) |
| Storing | `ragrails_store_chunks` | [Storing](/usage/mcp/storing) |
| Retrieval | `ragrails_retrieve` | [Retrieval](/usage/mcp/retrieval) |
