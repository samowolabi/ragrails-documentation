---
title: "MCP Embedding"
description: "Use MCP tools to embed chunk files."
---

`ragrails_embed_chunks` embeds chunk JSON files and writes vectors to the selected vector database.

Example agent request:

```text
Use ragrails_embed_chunks with input_dir files/output/chunks, vector_db qdrant, collection rag_chunks, embedder voyage, and model voyage-3.
```

Make sure the MCP server environment has provider credentials such as `VOYAGE_API_KEY`.
