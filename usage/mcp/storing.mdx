---
title: "MCP Storing"
description: "Use MCP tools to store chunks in a vector database."
---

`ragrails_store_chunks` is the explicit storage tool. It uses the same provider arguments as embedding.

Example agent request:

```text
Use ragrails_store_chunks with input_dir files/output/chunks, vector_db qdrant, collection rag_chunks, batch_size 64, embedder voyage, and model voyage-3.
```

Use storage extras such as `ragrails[store-qdrant]`, `ragrails[store-pinecone]`, or `ragrails[store-weaviate]`.
