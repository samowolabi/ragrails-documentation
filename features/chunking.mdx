---
title: "Chunking"
description: "Split markdown into retrieval-ready JSON chunks."
---

Chunking reads markdown output from ingestion and writes JSON chunk files for embedding and storage.

## Capabilities

- Split markdown using heading-aware structure.
- Preserve heading paths as chunk metadata.
- Generate stable chunk IDs from chunk content.
- Keep code blocks atomic so code is not split mid-block.
- Handle markdown tables as atomic or row-aware chunks.
- Repair links that would otherwise be split across chunk boundaries.
- Remove scraping navigation boilerplate when detected.
- Preview chunks for a single markdown file before running a full directory job.
