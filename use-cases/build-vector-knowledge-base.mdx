---
title: "Build a Vector Knowledge Base"
description: "Chunk, embed, and store source content in a vector database."
---

Use Ragrails to turn markdown outputs into a searchable vector collection.

## What you can do

- Chunk markdown files into JSON chunk files.
- Preserve title, source path, heading, and structural metadata.
- Embed all chunk files or selected files.
- Store vectors in Qdrant, Pinecone, or Weaviate.
- Use `store` when you want one explicit storage stage.

## Requirements

Choose the vector database and embedding provider you need:

```bash
pip install "ragrails[voyage,qdrant]"
docker run -p 6333:6333 qdrant/qdrant
```

## Usage

<Tabs>
  <Tab title="SDK">
    ```python
    from ragrails import RagRails

    rag = RagRails()

    chunks = rag.chunk(
        input_dir="files/output/docs",
        output_dir="files/output/chunks/docs",
    )

    stored = rag.store(
        input_dir=chunks.output_dir,
        vector_db="qdrant",
        collection="docs",
        url="http://localhost:6333",
        embedder="voyage",
        model="voyage-3",
    )
    ```
  </Tab>

  <Tab title="CLI">
    ```bash
    ragrails chunk \
      --input-dir files/output/docs \
      --output-dir files/output/chunks/docs

    ragrails store \
      --input-dir files/output/chunks/docs \
      --vector-db qdrant \
      --collection docs \
      --url http://localhost:6333 \
      --embedder voyage \
      --model voyage-3
    ```
  </Tab>

  <Tab title="REST API">
    ```bash
    curl -X POST http://127.0.0.1:8000/v1/chunk \
      -H "Content-Type: application/json" \
      -d '{
        "input_dir": "files/output/docs",
        "output_dir": "files/output/chunks/docs"
      }'

    curl -X POST http://127.0.0.1:8000/v1/store \
      -H "Content-Type: application/json" \
      -d '{
        "input_dir": "files/output/chunks/docs",
        "vector_db": "qdrant",
        "collection": "docs",
        "url": "http://localhost:6333",
        "embedder": "voyage",
        "model": "voyage-3"
      }'
    ```
  </Tab>

  <Tab title="MCP">

  </Tab>
</Tabs>
