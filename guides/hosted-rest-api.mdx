## Run the server

```bash REST API
pip install "ragrails[server-qdrant]"
ragrails-api
```

The server listens on `http://127.0.0.1:8000`.

| Resource | URL |
| --- | --- |
| Health check | `GET /v1/health` |
| Swagger UI | `GET /docs` |
| OpenAPI schema | `GET /v1/openapi.json` |

## Production checklist

- Provide provider credentials (`VOYAGE_API_KEY`, `OPENAI_API_KEY`, vector DB keys) as environment variables
- Put the service behind your API gateway, auth layer, and rate limiting
- Use a managed vector database (Qdrant Cloud, Pinecone, or Weaviate Cloud) for persistence
- Generate typed clients from `/v1/openapi.json`

See the [REST API reference](/reference/rest-api) for the full endpoint list.
