This example parses local documents, chunks them, embeds them, stores them in Qdrant, and retrieves results.

```bash
pip install "ragrails[voyage,qdrant]"
```

Start Qdrant locally before storing vectors:

```bash
docker run -p 6333:6333 qdrant/qdrant
```

```python
from ragrails import RagRails

rag = RagRails()

parsed = rag.parse(
    folder="files/input",
    output_dir="files/output/docs",
)

chunks = rag.chunk(
    input_dir=parsed.output_dir,
    output_dir="files/output/chunks/docs",
)

stored = rag.store(
    input_dir=chunks.output_dir,
    vector_db="qdrant",
    collection="rag_chunks",
    url="http://localhost:6333",
)

results = rag.retrieve(
    query="What does the documentation say about pricing?",
    vector_db="qdrant",
    collection="rag_chunks",
    url="http://localhost:6333",
    top_k=5,
)

print(results.results)
```

For URL ingestion, install `ragrails[url]` and run browser setup once:

```python
from ragrails import RagRails

rag = RagRails()
rag.setup_url()
```

Then use `rag.scrape(...)` before chunking.

For the same workflow through CLI, REST API, or MCP, use the tabs on the stage pages:
